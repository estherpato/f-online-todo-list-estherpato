### Guía de funcionalidad

Funcionalidades básicas:
* Añadir una tarea.
* Completar una tarea.
* Desmarcar una tarea completada.

### Especificaciones

* Realizado en *vanilla* JavaScript, sin librerías externas.
* Siempre muestra la fecha actual.
* Las tareas se guardan en el LocalStorage.
